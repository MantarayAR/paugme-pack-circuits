Circuits app for Paugme Packs

# Tests

Run tests using Gulp:

```
npm install -g gulp
npm install
gulp
```

Tests use [Jasmine](http://jasmine.github.io/), running on [Gulp-Jasmine](https://www.npmjs.com/package/gulp-jasmine).
